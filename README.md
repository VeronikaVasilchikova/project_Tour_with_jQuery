# project_Tour_with_jQuery

Create simple effects for modal window with jQuery:

1. choose three elements: "Выбрать тур", "Получить консультацию", "Расписание туров";
2. add to these elements event "click" which executes the appearance of elements with classes "overlay" and "modal"
3. choose an element with class "close"
4. add to these element event "click" which remove elements "overlay" and "modal"
